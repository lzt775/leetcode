# 二叉树最大路径和问题

## 问题描述

二叉树中的路径被定义为一条节点序列，序列中每对相邻节点之间都存在一条边，同一个节点在一条路径序列中至多出现一次，该路径至少包含一个节点，且不一定经过根节点。路径和是路径中各节点值的总和。给定一个二叉树的根节点 `root`，返回其最大路径和。

## 算法思路

这是一个经典的动态规划问题，使用递归的方式解决。核心思想是：

1. **递归计算每个节点的贡献值**：对于每个节点，我们计算以该节点为起点的单边最大路径和
2. **更新全局最大值**：在递归过程中，不断更新全局的最大路径和
3. **关键洞察**：一个节点的最大路径和可以是：
   - 只包含当前节点
   - 当前节点 + 左子树的单边最大路径和
   - 当前节点 + 右子树的单边最大路径和
   - 当前节点 + 左子树 + 右子树的单边最大路径和

## 算法步骤

1. 初始化全局最大值为负无穷
2. 定义递归函数 `max_gain(node)`：
   - 如果节点为空，返回0
   - 递归计算左右子树的单边最大路径和（如果为负则取0）
   - 计算包含当前节点的完整路径和
   - 更新全局最大值
   - 返回以当前节点为起点的单边最大路径和
3. 调用递归函数并返回全局最大值

## 时间复杂度

- **时间复杂度**: O(N)，其中 N 是树中节点的数量
- **空间复杂度**: O(H)，其中 H 是树的高度（递归调用栈的深度）

## 测试用例

### 示例 1
- **输入**: `root = [1,2,3]`
- **输出**: `6`
- **解释**: 最优路径是 `2 -> 1 -> 3`，路径和为 `2 + 1 + 3 = 6`

### 示例 2
- **输入**: `root = [-10,9,20,None,None,15,7]`
- **输出**: `42`
- **解释**: 最优路径是 `15 -> 20 -> 7`，路径和为 `15 + 20 + 7 = 42`

### 示例 3
- **输入**: `root = [2,-1]`
- **输出**: `2`
- **解释**: 最优路径只包含根节点 `2`

### 示例 4
- **输入**: `root = [1]`
- **输出**: `1`
- **解释**: 只有一个节点，最大路径和就是节点值

## 代码实现

```python
class Solution:
    def maxPathSum(self, root):
        self.max_sum = float('-inf')
        
        def max_gain(node):
            if not node:
                return 0
            
            left_gain = max(max_gain(node.left), 0)
            right_gain = max(max_gain(node.right), 0)
            
            price_newpath = node.val + left_gain + right_gain
            self.max_sum = max(self.max_sum, price_newpath)
            
            return node.val + max(left_gain, right_gain)
        
        max_gain(root)
        return self.max_sum
```

## 关键点

1. **单边路径 vs 完整路径**：
   - 递归函数返回的是单边最大路径和（只能选择左子树或右子树中的一个）
   - 全局最大值考虑的是完整路径（可以同时包含左右子树）

2. **负值处理**：
   - 如果子树的单边路径和为负值，我们选择0（即不选择该子树）
   - 这确保了路径和的最大化

3. **递归终止条件**：
   - 当节点为空时，返回0

4. **全局变量**：
   - 使用实例变量 `self.max_sum` 来在递归过程中更新全局最大值
